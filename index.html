Welcome to Standard Errors Page

Standard Errors is a project that aims to create a universal error standard for restful apis, database errors, form input errors, framework errors. to ease to way errors exchanging

Some Basics

  1. errors are identified by its code, each code is unique.
  2. errors can hold complicated information not just a message
  3. errors can be translated into every languages
  4. errors can be recursive and hierarchical
